# Sales Dashboard – Power BI Project

## Project Overview

This project analyzes sales data to identify trends, top-performing products, and customer behavior.
The dataset was cleaned and transformed before building an interactive dashboard in Power BI.

The goal is to provide a clear overview of business performance and support data-driven decision making.

---

## Tools Used

* Microsoft Excel – Data cleaning
* Power BI – Data modeling and dashboard creation

---

## Data Cleaning Process

The original dataset contained several issues:

* Mixed date formats
* Duplicate records
* Currency values stored as text
* Inconsistent formatting

The dataset was cleaned by:

* Standardizing date formats
* Removing duplicates
* Converting currency and numeric fields
* Creating calculated metrics

---

## Dashboard Features

Key metrics included:

* Total Sales
* Total Orders
* Total Customers
* Average Order Value

Visualizations:

* Sales trend by month
* Sales by city
* Top products by revenue
* Customer purchase behavior
* Sales distribution by category

Interactive filters allow users to explore data by:

* Customer
* Product
* Category
* Date range

---

## Key Insights

Some insights discovered in the analysis:

* Laptops generate the highest revenue among products
* Electronics account for most sales
* Buenos Aires and Córdoba lead in total sales
* A small number of customers generate a large portion of revenue
